# Mechanical Design

This folder contains the mechanical design files for the **AERIS-FSAR (Autonomous Fire Suppression and Response Robot)**.

The robot structure and component layout were designed using **Fusion 360** to ensure proper placement of motors, sensors, and the fire suppression system.

---

## Design File

`fusion360_model.f3d`

- Contains the complete 3D model of the robot chassis.
- Includes mounting positions for motors, sensors, battery, and fan.

---

## Design Considerations

- Compact rectangular chassis for stable movement.
- Central placement of the fan for effective fire suppression.
- Proper spacing for sensors to ensure accurate detection.
